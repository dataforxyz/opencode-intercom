# License Transition

This repository changed its current project license from MIT to the GNU Affero General
Public License v3.0 or later (`AGPL-3.0-or-later`) on 2026-07-14.

## Boundary

- Final MIT commit: [`d105070`](https://github.com/dataforxyz/agent-intercom-opencode/commit/d105070)
- Final MIT tag: [`mit-final`](https://github.com/dataforxyz/agent-intercom-opencode/tree/mit-final)
- First AGPL commit: [`56487b3`](https://github.com/dataforxyz/agent-intercom-opencode/commit/56487b3)
- First AGPL tag: [`agpl-transition`](https://github.com/dataforxyz/agent-intercom-opencode/tree/agpl-transition)
- First versioned AGPL release: `v0.3.0`

The transition is prospective. It does not revoke or alter rights granted for copies
already received under MIT. Current and future versions derived from the AGPL side of
the boundary are distributed under `AGPL-3.0-or-later`.

No earlier npm release is being retroactively changed. Git snapshots and packages built
from commits at or before `mit-final` remain under their original MIT terms.
